ODA最新补丁文档和下载地址

Falkon
2020-03-23
ODA, Patch
  • 登录官网链接http://www.oracle.com/goto/oda/docs
  • 点击Oracle Database Applicance Version Patchs前面的+号进入相关ODA一体机的补丁列表
  • 点击相应的补丁号，如27604623进入不行下载页面
  • 经测试ODA最新版本的补丁可以应用与较早的ODA一体机产品，如18.7的补丁可以应用早期的X6、X5以及X4，X3没有测试过
